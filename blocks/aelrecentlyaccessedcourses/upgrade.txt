This file describes API changes in the aelrecentlyaccessedcourses block code.

=== 3.8 ===

* New admin setting block_aelrecentlyaccessedcourses/displaycategories which enables the recently accessed
  courses block to display the course category.

=== 3.7 ===

* The 'block/aelrecentlyaccessedcourses:addinstance' capability has been removed. It has never been used in code.
